# Customer Segmentation App (Machine Learning + Streamlit)

This project is an interactive **Customer Segmentation Web App** built using **Python, Scikit-Learn, and Streamlit**.  
It predicts the customer segment based on demographic and purchasing behavior using a trained **K-Means Clustering** model.
